The Pipeline

EXTRACT · IMPORT · EDIT · EXPORT

Same four steps every time. The toolkit covers each one inside Blender.

01

Extract

RPKG-Tool pulls .prim, .borg, .MATI, and textures from the game's chunk archives.

02

Import

Load the files into Blender. Auto-Apply builds the full render material on import.

03

Edit

Reshape the mesh, repaint textures, swap materials, or replace the model entirely.

04

Export & Pack

Write the .prim, then pack into a patch .rpkg from the sidebar.

Import your own mesh with different topology and get it in-game with rigging and UVs intact. Two methods.

Method A — Custom Topology

Transfer weights, make exportable, export with Custom Mesh on. Bone-remap explosion bug fixed in v2.9.3.

Method B — Snap to Game Slot

Shrinkwrap projects the game mesh's verts onto your custom surface. Original vertex count and rigging stay untouched.
