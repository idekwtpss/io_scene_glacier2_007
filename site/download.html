Requirements

Blender

Version 4.2 or later. Earlier versions are not supported — the addon uses 4.2 APIs.

Operating System

Windows 64-bit. The native RPKG packer DLL (rpkg_native.dll) is Windows-only.

Game Files

007 First Light (PC). Use RPKG-Tool to extract .prim, .borg, .MATI, and texture files before importing.

RPKG-Tool

Required to extract game archives and deploy the exported patch .rpkg. Download separately from the RPKG-Tool project page.

Installation

1
Download the zip

Click Download v2.9.3 above. Save glacier2_007_toolkit.zip anywhere — do not extract it.

2
Open Blender Preferences

In Blender: Edit → Preferences → Add-ons → Install from Disk…

3
Select the zip

Navigate to glacier2_007_toolkit.zip and click Install Add-on.

4
Enable the addon

Search "Glacier" in the add-on list and tick the checkbox. Click Save Preferences.

5
Open the panel

In the 3D Viewport, press N → select the Glacier tab. You're ready to import.

Changelog

2.9.3

Current Release

  • FIX Bone-remap table corrected for custom topology exports — skinned meshes with different vertex counts no longer explode in-game. Previously all 255 remap slots were set to 0xFF (bone #255 / invalid), causing garbage GPU transforms. Now each used bone ID maps to itself.
  • NEW Snap Game Mesh to Custom Shape operator — projects game-mesh vertices onto a custom model's surface via Shrinkwrap. Keeps original vertex count and rigging intact; safest export path.
2.8.0

  • FIX Null-pointer crash from missing aux cull-cluster block on custom topology exports. Empty cluster block (type 0x20) now always written.
2.7.x

  • NEW RPKG packer integrated into the Export panel. External packer no longer required.
  • NEW Set Render Source — link a prim to its source RPKG chunk for correct hash generation.
2.6.x

  • NEW Cloth cage import / export. Conform Cloth to Active operator for re-targeting cloth rigs.
