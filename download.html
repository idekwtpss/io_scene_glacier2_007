Installation

1
Download the zip

Save glacier2_007_toolkit.zip anywhere. Don't extract it.

2
Install in Blender

Edit → Preferences → Add-ons → Install from Disk… → select the zip.

3
Enable

Search "Glacier", tick the checkbox, Save Preferences.

4
Open the panel

3D Viewport → N → Glacier tab.
